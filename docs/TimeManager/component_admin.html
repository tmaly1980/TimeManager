Description:

The Administration Console provides a way to manage user and group accounts, as well as products. It lets you assign groups to users, set up task categories, assign them to particular users, and define the products that tasks and projects are involved with. You can also edit your own profile, including personal information, preferences, and password.

Goals:

The Administration module helps you:

  • Organize people into teams for efficient management
  • Permit task assignment from one person to another in a non-hierarchical manager--worker fashion
Features:

The Administration module consists of the following major parts:

  • User Account Management: View and manage user accounts, their task category assignments, as well as browse a person's workload
  • Group Account Management: Create and modify groups, designate the manager, create task categories with stages, and grant users membership and management from the group's manager
  • Product Management: Designate the products within the company, along with revisions/versions for each
  • Workload Comparison: View and compare the workload of one or more people, and redistribute work if desired
Overview:

Here's what the module's major parts look like:

Administration Console:
The Administration Console contains the groups, users and products that you can manage, whether you're a site administrator or a manager.
User Administration:
Administer user accounts, specify task specializations, and explicit management rights.
Group Administration:
Specify group management, members, and task categories.
Product Administration:
Specify products with their versions, to associate with tasks.
Task Category Specification:
Task categories are defined for each group of users. The task categories may have stages, further describing the exact progress level.
User/Category Association:
You can define which task categories a user can and cannot do via the user edit page. By default, users can perform newly added task categories.
Workload Comparison:
Compare and redistribute the workload for your team's members.
Getting There:
Administration Console:
Click on the "Admin" tab to get to the Administration Console. This menu is only available for site administrators and people who are either group managers or can manage (assign tasks to) other users.
My Profile:
Edit your own profile, including personal information, password, and preference to have the power-user interface by default on tasks (having all sections expanded when opening a task).
