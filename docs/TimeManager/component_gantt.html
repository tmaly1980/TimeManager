Time Manager: Gantt Charts

The Gantt Chart module provides a way to view a graphical timeline snapshot of all the tasks involved. It gives you a fairly quick idea as to how things are going, and lets you view the details of individual tasks if needed.

Goals:

The Gantt Charts module helps you:

  • Monitor the progress of a project graphically
Overview:
Gantt Chart:
Features:

The Gantt Charts module consists of the following major parts (click on the name for further details):

  • Date Span: Being able to zoom in or out to fit the exact given dates on the graph
  • Zoom to Fit: Zooming to fit the entire timeframe of all tasks within the window
  • Ruler: Show the metrics of what equates to a certain unit of time
  • Print: Print the Gantt chart for later use, such as at meetings
  • Timeline bars: Display the entire timeframe of a task in a rectangular box, filled in green to mark progression, as well as having the percent completed listed. Canceled tasks show up in red.
  • Task Viewing/Editing: Select a task to go to the view/edit page for further follow-up, such as perhaps to adjust to meet criteria for overall goal
Getting There:
Parent Task Gantt:
Gantt charts can be displayed for parent tasks containing dependencies and subtasks.
Task Group Gantt:
Task Groups and Milestones can display Gantt charts for the containing tasks.
Project Gantt:
Projects can utilize Gantt charts to display every milestone and task within it.
