Time Manager: Help System
  • Description
  • Goals
  • Overview
Description:

The Integrated Help System module provides a way to integrate the documentation (what you're looking at now) with the application. In essence, it provides a way to learn things as you need them. A 'HELP' button is provided anywhere that context-specific help is available.

Goals:

The Help System module helps you:

  • Learn task/project management as you go, with a context-sensitive help system
Overview:

Here's what the module's major parts look like:

Task Search Help:
Help links for task search page.
Project Help:
Help links for project page.
Add Task Help:
Help links for adding a task.
Admin Page Help:
Help links for user, group and product administration.
