Time Manager: Logging/Notes

The Logging Module provides a way to keep track of any arbitrary amount of notes, including automatically logging significant changes in a task for future reference, such as the task's owner, title, due date, estimated completion date, status, priority, requestor, task category, and hours. Each note contains a timestamp and names the person who put the note there.

Goals:

The Logging/Notes module helps you:

  • Keep track and reference when task specifications and progress changes and why
  • Keep track of all people involved in a task throughout it's lifetime
Overview:
Task Notes:
Each task contains a section to add and display notes, including automatic logging of changes.
Features:

The Logging/Notes module consists of the following major parts (click on the name for further details):

  • Notes: Enter in arbitrary notes to keep track of information or thoughts related to a task
  • Logging: Keep track of major changes to a task,such as the task's owner, title, due date, estimated completion date, status, priority, requestor, task category, and hours
Getting There:
Task Edit Page:
Logging of vital information and keeping track of notes is provided via the task edit page.
