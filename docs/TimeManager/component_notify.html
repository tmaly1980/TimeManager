Time Manager: Notification

The Notification Module provides a means to keep people related to a task informed when vital information changes, such as when a task is created, completed, or re-prioritized.

  • Goals
  • Overview
  • Features
    • Task Owner Notification
    • Task Requestor Notification
Goals:

The Notification module helps you:

  • Keep task owners and requestors automatically notified via email when a task changes
Overview:
Notification Email:
Email recieved when a task changes, including vital information as to what has changed near the top, with a snapshot of the requirements and status, along with the notes.
Features:

The Notification module consists of the following major parts (click on the name for further details):

  • Task Requestor Notification: Keep a task's requestor notified via email when the person responsible changes vital information such as the status, percent completed, and cost.
  • Task Owner Notification: Keep a task owner notified via email when the task requestor creates or changes a task
