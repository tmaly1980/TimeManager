Description:

The Product Management module provides a way to get a large set of tasks done in an organized project. It helps you take an idea, turn it into separate milestones, and then translate those milestones into tasks distributed to others. It also helps you keep track of how well a project is progressing.

Goals:

The Project Management module helps you:

  • Schedule, readjust and keep projects on time
  • Organize projects without too much formality or complexity
Features:

The Project Management module consists of the following major parts:

  • Project Specification: Define what a project is and what you expect out of it, including dates, budget, priority and resources.
  • Project Overview Panel: View a quick snapshot of the overall specifications and status of a project, including the status of the milestones.
  • Milestones / Task Groups: Designate goals and major markers within a project, and subdivide each of them into groups of tasks.
  • Project Searching: Find the projects you want to work and check up on.
  • Participant List: Get others to help you complete a project, by working on the tasks
  • Browse Panel: Quickly see what you or your managed workers are working on, with a brief status.
  • Gantt Chart: View a graphical timeline with status of the tasks in a project.
Overview:

Here's what the module's major parts look like:

Project Overview:
Get a quick snapshot of the requirements, status and milestones of a project.
Milestone Editing:
View and edit a milestone, along with the tasks it contains, whether all or only the remaining ones to complete.
Gantt Chart:
Get a visual overview of how tasks are progressing within the overall project.
Project Search:
Find projects you or others are working on.
Getting There:
Project Management Console:
To get to the Project Management Console, click on the "Projects" tab from the main window.
Project Browse Panel:
To view a list of current projects from within the Project Management Console that you or a managed worker is working on, click on the "Browse" button immediately below the "Projects" tab.
Project Overview:
To show the overview of a project, including it's requirements, participants, and status, click on the name of the project from either the Project Browse page, or the Project Search page.
Project Search:
To get to the project search page, click on the "Search" button within the Project Management Console, fill out the form, and then click on "Search" to view the results.
Project Edit Page:
To edit the specifications and status of a project, click on the "Edit" link from the Project Overview page.
Project Add Page:
To add a new project, click on the "Add Project" navigational link from anywhere within the Project Management Console.
Milestone Edit Page:
From the Project Overview page, click on a milestone's name (or ID) to open the edit window for that milestone.
Gantt Chart:
To view the timeline Gantt chart, click on "Gantt" from the Project Overview page.
