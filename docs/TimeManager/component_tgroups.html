Time Manager: Task Groups/Milestones

Task Groups and Milestones provide a way to organize tasks into relevent groups having a common goal. Grouping is arbitrary, and tasks can belong to multiple task groups. It helps you keep track of things on a broader scale than at the task level, while more refined than at the project level. Milestones are a special kind of task group that are tied to a specific project.

Goals:

The Task Groups/Milestones module helps you:

  • Subdivide goals or milestones of a project into groups of tasks
  • Track the overall goal of a set of tasks at a much broader level
Overview:
Task Group Editing:
The window where you can edit and oversee the progress of the task group's goal. Relevent information from tasks are summarized.
Features:

The Task Groups/Milestones module consists of the following major parts (click on the name for further details):

  • Summary / Description: Provide a concise as well as a descriptive text as to what the task group's goal is
  • Priority: Provide an easy way to compare the importance of different task groups
  • Owner: Give management of the task group to an individual, responsible for ensuring the goal is met
  • Start / Due Dates: Specify the start date and due date that you want the tasks to be confined within
  • Status, Percent, Hours: Automatically calculate the status, percent completed, and actual versus estimated hours from the list of tasks within the task group
  • Notes: Keep arbitrary information in the 'Notes' section, such as progress, thoughts, and reasons for delays, etc.
  • Gantt Chart: Display a Gantt Chart for the task group / milestone and the contained tasks, for a quick graphical assessment as to progress
  • Bulk Changes: Organize and change the specifications, timeline and status of multiple task groups at a time, so as to handle generalized reassignment, project delays, and project redesign.
Getting There:
Adding Independent Task Groups:
From the Task Group Console, you can add independent task groups by the 'Add Task Group' link.
Linking New Task Groups to Tasks:
From within a task, you can specify a new task group to link to it.
Linking Existing Task Groups:
You can also link an existing task group to a task.
Project Milestones:
Milestones are shown in a concise manner on a project's overview page. Clicking on their name goes to the actual edit screen.
Adding Milestones to Projects:
You can add existing or new milestones to projects, through the project overview page.
