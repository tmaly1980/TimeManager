Time Manager: Workload Management

The Workload Management Console provides a means to monitor how much work (displayed as hours allocated versus hours available) that one or more workers have over a period of any number of weeks. The console also provides a means to redistribute any tasks that you want redistributed.

Workload Comparison:
Features:

The Workload Management module consists of the following major parts (click on the name for further details):

  • Workload Comparison: Compare the workload between any number of people
  • Workload Trends: See how one or more person's workload changes over time, to know when to best assign and redistribute tasks
  • Workload Redistribution: After comparing the workload of multiple people over multiple weeks, quickly redistribute tasks for maximum efficiency
Getting There:
Administrative Workload Comparison:
From the Administration Console, a manager can select (via checkboxes) which people that they manage, and simultaneously view their workload (via the "View/Compare Workload" button).
Task Assignment Workload Comparison:
From a task, the task requestor can select the "Workload" button next to the owner drop-down list to either see the current owner's workload is (if a current owner is specified), or to compare the workload of all potential owners (if no owner is specified).
