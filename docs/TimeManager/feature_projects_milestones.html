Time Manager: Task Management: Milestones / Task Groups
Description:

Task Groups and Milestones provide a way to organize tasks into relevent groups having a common goal. Grouping is arbitrary, and tasks can belong to multiple task groups. It helps you keep track of things on a broader scale than at the task level, while more refined than at the project level. Milestones are a special kind of task group that are tied to a specific project.

Items:

This feature consists of the following items (marked on the overview images by the respective number):

  1. Summary / Description: Provide a concise as well as a descriptive text as to what the task group's goal is.
  2. Priority: Provide an easy way to compare the importance of different task groups / milestones.
  3. Owner: Give management of the task group to an individual, responsible for ensuring the goal is met.
  4. Start / Due Dates: Specify the start date and due date that you want the tasks to be confined within.
  5. Status, Percent, Hours: Automatically calculate the status, percent completed, and actual versus estimated hours from the list of tasks within the task group.
  6. Notes: Keep arbitrary information in the 'Notes' section, such as progress, thoughts, and reasons for delays, etc.
  7. Gantt Chart: Display a Gantt Chart for the task group / milestone and the contained tasks, for a quick graphical assessment as to progress.
  8. Bulk Changes: Easily organize task groups/milestones as well as set and change the specifications and status of as many task groups at a single time, including changing the owner, project, priority and start and due dates.
