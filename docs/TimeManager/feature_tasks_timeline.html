Time Manager: Task Management: Timeline
Description:

Items:

This feature consists of the following items (marked on the overview images by the respective number):

  1. Start Date: When work on a task should begin.
  2. Due Date: The desired date (as set by the task's requestor) in which the work of a task should be completed.
  3. Estimated Finish Date: The estimated date (as set by the task owner) as to when the task should be completed.
  4. Actual Finish Date: Once a task is completed, the actual finish date is automatically set to that day.
  5. Estimated Hours: How many hours that a task is espected to take. It is typically filled out by the task requestor.
  6. Actual Hours: How many hours a task really took to complete.
  7. Calendar Button: A button designating that the date next to it should be entered in via a visual calendar window.
  8. "Today" Button: A button that designates that the date field next to it should be set to today's date.

Usage:
Setting Dates:
Dates must be put in YYYY-MM-DD format, such as 2004-05-31 for May 31st, 2004.
Accessing the Calendar:
Click on the 'Calendar' button immediately below the respective date field.
Using the Calendar:
With the Calendar open, you can move forward or backward in month or year increments by using the single or double arrows, respectively. You can return to the current month by clicking on the month name displayed.
