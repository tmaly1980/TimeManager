Introduction

Mission Statement:

Provide task and project management, especially for the software and IT fields, in a manner that provides convenience, rapid organization, and rapid completion, yet without the inconvenience, complexities and delays involved with strict and formal project management.

Time Manager helps you:

With the Dashboard module:
  • Monitor tasks and projects that you and your team have due
With the Task Management module:
  • Find tasks that you have time available for
  • Assign, manage, and monitor tasks and projects to your team members
  • View and redistribute your team's workload via graphs
  • Manage subtasks and dependencies
  • Perform changes to multiple tasks simultaneously
  • Keep track and manage a rapidly dynamic queue of tasks
  • Eliminate redundant work by referencing existing tasks
  • Keep track of how much time is spent on certain types of tasks, to justify hiring dedicated workers or increased budgets
  • Find tasks based upon specifications, timeline, status, participants or text
With the Task Groups/Milestones module:
  • Subdivide goals or milestones of a project into groups of tasks
  • Track the overall goal of a set of tasks at a much broader level
With the Project Management module:
  • Schedule, readjust and keep projects on time
  • Organize projects without too much formality or complexity
With the Gantt Charts module:
  • Monitor the progress of a project graphically
With the Task Specialization module:
  • Figure out who should do a task based upon what it involves
With the Workload Management module:
  • Reassign tasks to avoid inefficiency and optimize appropriate assignment settings
With the Notification module:
  • Keep task owners and requestors automatically notified via email when a task changes
With the Logging/Notes module:
  • Keep track and reference when task specifications and progress changes and why
  • Keep track of all people involved in a task throughout it's lifetime
With the Administration module:
  • Organize people into teams for efficient management
  • Permit task assignment from one person to another in a non-hierarchical manager--worker fashion
With the Security module:
  • Enforce allowing people to view and edit only the projects and tasks that they're supposed to
  • Enforce task assignment in a hierarchical structure
  • Enforce the proper requirements and specifications of a task to be defined only by the task's requestor
  • Enable managers other than the task's requestor to see your tasks, but not alter them
With the Help System module:
  • Learn task/project management as you go, with a context-sensitive help system

Audience

  • IT Managers in a rapidly growing company, needing to organize and scale IT operations
  • Project Managers who want the benefits of project management to ensure success yet without the delays and complexities, to ensure rapid completion
  • Software Engineers who want to develop and promote quality and rapidly developed Open Source software, with only their spare time available
  • IT Help Desk personnel who want to focus on one task at a time, without losing track of their growing queue
  • Multi-hatted Software Engineers who simultaneously need to work on several components of one or more products, possibly working in multiple roles such as design, development, and testing, while keeping organized and focused

System Requirements

  • Web Server: Apache 1.3 Web Server or higher, or equivalent
  • Perl: Perl 5.6.1 interpreter or higher
  • Relational SQL Database Server: MySQL 3.23+ Database Server, or equivalent SQL-compliant Database Server
  • UNIX/Linux Operating System: Linux 2.4 or other UNIX-based Operating System
  • Web Browser: Internet Explorer 5.5+ or Netscape 7.0+, or equivalent web browser
  • JavaScript: JavaScript enabled in web browser
