Product Tutorial: Editing a Project

After adding a project, one can then edit the project to define milestones, view the Gantt chart, and more.

  1. Overview section:

    2. The 'overview' section displays a summary of the project, as altered via the edit menu.

  2. Edit Overview:

    3. By clicking on the 'Edit' link near the top, one can go to a screen to edit the details of the project.

    See Also:
    • Adding a Project
  3. Adding a new milestone:

    4. To add a new milestone, click on the 'Add New' link next to the 'Milestones' header. This will open a new window for you to fill out. After saving and closing the window, the project window should refresh.

    See Also:
    • Adding Task Groups
  4. Adding an existing milestone:

    5. By clicking on 'Add Existing', one gets a search form to specify the criteria to find the desired milestones. After clicking on 'Link', and closing the window, one should see the updated project with the newly linked milestone.

  5. View Active Tasks:

    6. By clicking on 'View Active Tasks', one is able to see and alter any tasks that yet to be finished fo the project to be complete.

  6. Gantt Chart:

    7. By clicking on the 'Gantt' button, one gets a Gantt chart for the whole project.

    See Also:
    • Using Gantt Charts
  7. Canceling the Project:

    8. If a project should be canceled, you can do so by clicking on the 'CANCEL THIS PROJECT' button on the edit screen. It will prompt for verification.
