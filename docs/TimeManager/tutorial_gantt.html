Product Tutorial: Using Gantt Charts

The Gantt chart shows a timeline-based graphical breakdown and summary of a project (or group of tasks). It can both provide a quick understanding as to the progress, or can point out any causes of delays/problems.

Sections:

  1. Opening the Gantt from a project
  2. Opening the Gantt from a task group/milestone
  3. Opening the Gantt from a parent task
  4. Entering in dates
  5. Zoom to Fit
  6. Print
  7. Clicking on a task / milestone
  8. Timeline Ruler
  1. Opening the Gantt from a project:

    2. By clicking on 'Gantt' from within a project overview, one can then get to the graphical window for that project.

  2. Opening the Gantt from a task group/milestone:

    3. Simply click on 'Gantt' from within an edit page for a task group / milestone.

  3. Opening the Gantt from a parent task:

    4. By clicking on 'Gantt' next to the Subtasks/task dependencies section in a task, one can get a Gantt chart of the parent task and all dependencies/subtasks.

  4. Entering in dates:

    5. By entering in dates and refreshing the graph, one gets a more refined display for exactly those dates. This becomes very useful when projects become months long and you just want to know what's going on this week.

  5. Zoom to Fit:

    6. 'Zoom to fit' provides a way to fit the entire set of tasks (i.e. project) within the screen. It helps you see everything at once.

  6. Print:

    7. Clicking on 'Print' creates a printable version of the Gantt chart.

  7. Clicking on a task / milestone:

    8. By clicking on any item on the left, one is provided with the details screen of that item.

  8. Timeline Ruler:

    9. The ruler shows how many screen pixels equate to days/weeks. The ruler repeats every 10 rows of data, so as to allow for scrolling yet being able to read/estimate the dates of tasks based upon the progress meter bar.
