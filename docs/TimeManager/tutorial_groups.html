Product Tutorial: Managing Group Accounts

The Administration Console provides a way to add and edit groups, as a site administrator or manager.

Sections:

  1. Adding a Group
  2. Editing a Group
  3. Editing Task Categories
  4. Editing Task Category Stages
  5. Members
  1. Adding a Group:

    2. Adding a group is done by clicking on the 'Add' link from the main Admin menu.

  2. Editing a Group:

    3. Editing a group is done by clicking on the group's name from the admin menu.

  3. Editing Task Categories:

    4. Task categories specify what kind of tasks can be done. You can add to or replace entries in the list by typing in a name in the text field box and clicking on 'Add' or 'Replace', respectively. Removing a task category is done by highlighting the category and then clicking on 'Remove'.

  4. Editing Task Category Stages:

    5. Stages specify levels of progress, specific to the task's category. You can alter the stages for a given task category by highlighting the task category, editing the right-hand list (using the Up/Down navigation for ordering), and clicking on '<< Save' (in between the two boxes) to save the list back. Changes don't take effect until the 'Save' button at the bottom is clicked on.

  5. Members:

    6. One can add or remove members by using the left or right arrows, respectively (after selecting an appropriate name to affect).
