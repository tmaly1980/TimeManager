Product Tutorial: Managing Products

Products are listed and managed from within the Administration Console. Products can only be added and editing by a site administrator.

Sections:

  1. Versions
  1. Versions:

    2. From within a product's edit page, on can specify the versions belonging to a product. Each version contains a numerical version (such as 1.0, 3.1.1), but can also contain a name (such as 'Enterprise Edition', 'Gamma Version'). One puts in such information in the text boxes, then clicks on 'Add' or 'Replace' to put into the list.
